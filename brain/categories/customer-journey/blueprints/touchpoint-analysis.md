# Touchpoint Analysis Board

## Overview

- **Purpose**: A comprehensive touchpoint matrix that maps every customer interaction across channels (rows) and journey stages (columns), with satisfaction scores, ownership assignments, and improvement opportunities for each intersection. This board reveals where the customer experience is strong, where it breaks down, and where critical gaps exist — enabling teams to prioritize investments with surgical precision.
- **Best For**: CX directors auditing omnichannel experience quality, product and marketing teams identifying underperforming touchpoints, and operations leaders rationalizing channel strategy and resource allocation.
- **Complexity**: Advanced
- **Board Size**: 5000 x 3500px

## Board Layout

The board is structured as a large matrix with channels as rows and journey stages as columns. Each cell contains the touchpoint details, a satisfaction score, and an improvement note. A summary row at the bottom shows stage-level scores. A summary column on the right shows channel-level scores. A priority action panel sits below the matrix.

## Variations

1. **Channel Deep-Dive**: Take one channel (e.g., Email) and break it into sub-touchpoints (welcome email, trial drip, renewal notice, etc.) with detailed metrics per email.
2. **Competitor Comparison**: Add a second matrix showing competitor touchpoint quality side by side. Identify where you lead and where you lag.
3. **Internal Stakeholder Map**: Replace satisfaction scores with owner assignments. Use the matrix to clarify which team owns each touchpoint and identify orphaned touchpoints with no clear owner.
4. **Quarterly Tracking**: Create the same matrix for Q1, Q2, Q3, Q4. Track score improvements over time as a CX scoreboard.
5. **B2C Version**: Replace channels with consumer-relevant ones (Mobile App, Physical Store, Packaging, Delivery, Social, Email). Replace journey stages with consumer stages (Browse, Cart, Checkout, Unbox, Use, Re-order).

## Build Instructions

1. Create board at 5000 x 3500px with Cloud White (#F5F5F5) background.
2. Place the title banner (4900 x 300px) at the top with Ocean Blue background.
3. Build the matrix structure: 6 column headers (stages) + 7 row headers (channels) = 42 cells.
4. Create column headers as rectangles with white text on Ocean Blue.
5. Create row headers as rectangles with Ocean Blue text on Soft Sky background.
6. For each active cell, add a satisfaction score circle (color-coded: green/yellow/red), a descriptive sticky note, and an opportunity/pain sticky note.
7. Mark inactive cells as "N/A" with grey fill.
8. Add the Stage Average row at the bottom of the matrix with large score circles.
9. Create the Heatmap Summary (1500 x 600px) at bottom-left with a mini color-coded matrix.
10. Create Priority Improvements (1800 x 600px) at bottom-center with gold-tinted background.
11. Create Gap Analysis (1400 x 600px) at bottom-right with coral-tinted background.
12. Draw flow arrows across stages, highlight borders around extreme scores, and connect red cells to priority improvements.
13. Lock matrix structure, headers, and scores. Leave improvement notes and gap analysis sticky notes editable.

## Tips & Best Practices

- Populate scores with real data — NPS, CSAT, session recordings, and support ticket analysis. Subjective scoring leads to politics, not insights.
- The heatmap view is the most powerful communication tool. Executives see the red zones instantly and understand the priorities.
- Assign clear ownership for every active touchpoint. If nobody owns it, nobody improves it.
- N/A cells are not inherently bad — not every channel needs to be active at every stage. But every N/A should be a conscious decision, not an oversight.
- Review the matrix quarterly with cross-functional stakeholders. Each team sees only their channel; the matrix reveals the full picture.
- Use this board alongside the Customer Journey Map. The journey map shows the user's experience; the touchpoint analysis shows the operational reality behind it.
- Start with the lowest-scoring stage (often Purchase or Onboarding) and work outward. Fixing the weakest link has the highest impact on overall CX.
- Celebrate improvements. When a cell moves from red to yellow or yellow to green, highlight it in the next review meeting.
