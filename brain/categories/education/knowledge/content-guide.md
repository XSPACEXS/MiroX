# Content Guide — Education Boards

## Overview

Education board content must serve both planning and communication. A lesson plan board plans the session for the teacher AND communicates expectations to observers. A curriculum board plans the semester AND demonstrates pedagogical rigor to reviewers. The content must be precise, pedagogically sound, and rich enough to actually teach from.

## Essential Sections by Board Type

### Lesson Plan — Required Sections

| Section               | Purpose                          | Content Type                                      |
| --------------------- | -------------------------------- | ------------------------------------------------- |
| Learning Objectives   | What students will know/do       | 2-4 measurable objectives with Bloom's verbs      |
| Standards Alignment   | Curriculum compliance            | Standards codes with descriptions                 |
| Prior Knowledge       | What students must already know  | Prerequisites and assumed skills                  |
| Materials & Resources | What is needed to teach          | Physical items, digital resources, prep checklist |
| Activity Timeline     | Minute-by-minute class flow      | Sequenced activities with duration, instructions  |
| Teacher Notes         | Facilitation guidance            | Key questions, transitions, timing cues           |
| Assessment            | How to measure learning          | Formative checks, exit tickets, rubric criteria   |
| Differentiation       | Adaptations for diverse learners | Scaffold, core, and extension for each activity   |

### Research Board — Required Sections

| Section           | Purpose                         | Content Type                                      |
| ----------------- | ------------------------------- | ------------------------------------------------- |
| Research Question | The driving inquiry             | Clear, specific, researchable question            |
| Hypothesis        | Predicted outcome               | Testable prediction with variables identified     |
| Literature Review | Context of existing knowledge   | 8-15 key sources with findings and gaps           |
| Methodology       | Research approach               | Design, sample, instruments, procedures, validity |
| Data Collection   | How data will be gathered       | Timeline, tools, ethical considerations           |
| Analysis Plan     | How data will be analyzed       | Statistical methods, software, thresholds         |
| Findings          | Results (if available)          | Data presentations, statistical results           |
| Discussion        | Interpretation and implications | Meaning, limitations, future research             |
| References        | Source documentation            | APA/MLA formatted citations                       |

## Real Content Examples

### Example 1: Lesson Plan — Grade 9 Biology

**Learning Objectives**:

1. LO1 (Understand): Describe the overall process of photosynthesis in a single sentence, including inputs (CO2, H2O, light) and outputs (glucose, O2).
2. LO2 (Apply): Label the key structures in a leaf cross-section diagram (chloroplast, stroma, thylakoid, stomata) and explain each structure's role.
3. LO3 (Analyze): Predict what would happen to a plant's glucose production if one input (light, water, or CO2) were removed, and justify the prediction using the photosynthesis equation.

**Standards**: NGSS LS1.C, Common Core RST.9-10.7

**Activity Timeline** (50 minutes):

| Time      | Activity                                                                                | Type          | Bloom's            |
| --------- | --------------------------------------------------------------------------------------- | ------------- | ------------------ |
| 0-5 min   | Opening Hook: "Why are plants green?" — Students write a hypothesis on mini-whiteboards | Engagement    | Remember           |
| 5-15 min  | Direct Instruction: Photosynthesis equation, inputs/outputs, with Elodea demo           | Input         | Understand         |
| 15-20 min | Pair Activity: Label the leaf cross-section diagram using colored pencils               | Practice      | Apply              |
| 20-30 min | Group Lab: Elodea + bromothymol blue + light — observe color change                     | Investigation | Apply              |
| 30-40 min | Class Discussion: "What happened? Why?" — Connect observation to equation               | Analysis      | Analyze            |
| 40-45 min | Prediction Challenge: "What if we covered the plant?" — Individual written response     | Assessment    | Analyze            |
| 45-50 min | Exit Ticket: 3 questions (Google Form) + Closing Reflection                             | Assessment    | Understand/Analyze |

**Teacher Notes for Direct Instruction**:

- "Cue the video to 0:32 — skip the intro"
- "When explaining the equation, write it as a word equation FIRST, then chemical symbols"
- "Common misconception: students think plants 'breathe' CO2 like we breathe O2 — clarify that photosynthesis is a chemical process, not 'breathing'"
- "Pause after the Elodea demo to ask: 'What do you notice about the bubbles?'"

**Differentiation — Labeling Activity**:

- Scaffold: "Provide a word bank with structure names and function descriptions. Students match rather than recall."
- Core: "Students label the diagram using their textbook as reference."
- Extension: "Students label without references AND draw arrows showing the flow of energy through the leaf."

### Example 3: Research Board — Education Technology Study

**Research Question**: "How does the integration of AI-powered adaptive learning platforms affect math achievement scores among 10th-grade students in Title I schools, compared to traditional instruction?"

**Hypothesis**: "Students using AI-adaptive platforms will show significantly higher gains on standardized math assessments (p < .05) compared to a control group receiving traditional instruction, after controlling for prior achievement, socioeconomic status, and teacher experience."

**Literature Review** (key sources):

1. Johnson & Lee (2024): AI tutoring improved algebra scores by 0.4 SD in 8th grade (n=1,200)
2. Patel et al. (2023): Adaptive learning showed no significant effect in low-income schools — implementation fidelity was the key factor
3. Rodriguez (2025): Teacher attitudes toward AI tools predicted student engagement more than the tools themselves
4. Meta-analysis: Chen & Wu (2024): Effect size 0.3 SD across 45 studies, but publication bias concerns
5. **GAP**: No study has specifically examined AI-adaptive platforms in Title I high schools with implementation fidelity controls

**Methodology**:

- Design: Quasi-experimental, pre-post with matched control group
- Sample: 8 Title I high schools, ~400 students (200 treatment, 200 control)
- Treatment: AI-adaptive platform (DreamBox Math) integrated into daily instruction for 16 weeks
- Control: Traditional instruction with same content standards
- Measures: Pre/post state standardized math assessment + weekly platform usage data
- Analysis: ANCOVA controlling for pre-test scores, SES, and teacher experience

## Copywriting Principles for Education

### 1. Write Objectives as Observable Behaviors

**Bad**: "Students will understand photosynthesis."
**Good**: "Students will describe the overall process of photosynthesis in a single sentence, including inputs (CO2, H2O, light) and outputs (glucose, O2)."

"Understand" is invisible. "Describe in a single sentence" is observable and measurable.

### 2. Use Bloom's Verbs Deliberately

Each cognitive level has specific action verbs:

- Remember: List, define, recall, identify, name
- Understand: Describe, explain, summarize, paraphrase, classify
- Apply: Demonstrate, solve, use, implement, calculate
- Analyze: Compare, contrast, differentiate, examine, categorize
- Evaluate: Judge, assess, critique, justify, defend
- Create: Design, construct, develop, formulate, compose

### 3. Include Anticipated Student Responses

**Activity description**: "Ask students: 'Why are plants green?'"
**With anticipated response**: "Ask students: 'Why are plants green?' Expected responses: 'Because of chlorophyll' (partial — push further), 'Because they reflect green light and absorb other colors' (strong), 'Because of photosynthesis' (common misconception — photosynthesis uses light, not color)."

Anticipated responses help teachers prepare for real-time facilitation.

### 4. Teacher Notes Should Sound Like a Coach

**Bad**: "Transition to the next activity."
**Good**: "Before moving to the lab, check for understanding with a quick thumb vote. If more than 5 students are 'thumb down,' re-explain the equation using the simple version before proceeding."

Teacher notes should be practical, specific, and situation-aware.

### 5. Time Estimates Should Be Realistic

Add buffer time annotations:

- "This activity says 10 minutes, but in my experience it takes 12-15. Shorten the closing reflection if needed."
- "If the Elodea demo takes longer than 5 minutes to show results, have a backup video cued up."

## Common Content Mistakes

### 1. Vague Objectives

"Students will learn about the Civil War" is not an objective. It is a topic. Objectives must include an observable verb, the specific knowledge or skill, and the success criteria.

### 2. Activity Without Purpose

Every activity should connect to a learning objective. If you cannot explain which objective an activity serves, it may be filler. Remove it or find a better alternative.

### 3. Assessment Misalignment

If the objective says "analyze" but the assessment asks students to "list," there is an alignment gap. The cognitive level of the assessment must match the cognitive level of the objective.

### 4. One-Size-Fits-All

A lesson plan without differentiation assumes all students are at the same level. Include at least scaffold and extension options for the main activities.

### 5. No Time for Reflection

A lesson that fills every minute with instruction and activity but leaves no time for student reflection misses a critical learning step. Always include 3-5 minutes for students to process what they learned.

## Content Templates

### Learning Objective Template

```
Students will be able to [Bloom's verb] [specific knowledge/skill]
[conditions/context] [success criteria].

Example: Students will be able to [predict] [what would happen to glucose
production if one input were removed] [given the photosynthesis equation]
[with correct identification of the dependent variable and logical justification].
```

### Activity Block Template

```
Activity Name: [Name]
Duration: [X minutes]
Type: [Input / Active / Assessment / Reflection]
Bloom's Level: [Remember through Create]
Objective Alignment: [Which LO(s) this addresses]
Instructions: [What students do, step by step]
Materials: [What is needed]
Teacher Role: [What the teacher does during this activity]
Differentiation:
  - Scaffold: [Simplified version for struggling learners]
  - Extension: [Advanced version for ready learners]
Assessment Check: [How to know students are succeeding during this activity]
```

### Research Question Template

```
Research Question: "How does [independent variable] affect [dependent variable]
in [population/context], compared to [comparison/control]?"

Variables:
  - Independent: [What is being manipulated or examined]
  - Dependent: [What is being measured]
  - Control: [What is held constant]
  - Confounding: [What might interfere — and how it is controlled]
```
