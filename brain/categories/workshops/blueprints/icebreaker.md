# Team Icebreaker Board

## Overview

- **Purpose**: A collection of fun, low-pressure icebreaker activities designed to build rapport, create psychological safety, and energize teams before workshops, offsites, or project kickoffs. The board includes multiple activity zones that can be used independently or as a sequence, giving facilitators flexibility to match the energy level and time available.
- **Best For**: Team managers onboarding new members, facilitators warming up groups before workshops, and leaders kicking off offsites or all-hands meetings with 4-30 participants.
- **Complexity**: Simple
- **Board Size**: 3000 x 2500px

## Board Layout

The board is divided into 4 distinct activity zones, each self-contained. A welcome banner sits at the top. Teams can navigate to whichever activity the facilitator chooses or work through multiple activities sequentially.

```
+------------------------------------------------------------------+
| [ WELCOME BANNER — "Let's Break the Ice!" ]                       |
+------------------------------------------------------------------+
|                                                                    |
|  +---------------------+    +------------------------------+      |
|  | FUN QUESTION         |    | PERSONAL FACT CARDS          |      |
|  | PROMPTS              |    | ("About Me" mini-posters)    |      |
```

## Variations

1. **Remote-First Version**: Make Photo Gallery the primary activity (async-friendly). Add a "Virtual Background Challenge" — share the story behind your Zoom background.
2. **Large Group (30+)**: Use Fun Questions in breakout groups of 5-6. Skip Two Truths (too slow for large groups). Use Photo Gallery as a passive browse-at-your-pace activity.
3. **Recurring Team Ritual**: Rotate one icebreaker per weekly meeting. Monday: Fun Question of the Week. Keep a running "Team Trivia" board of fun facts learned over time.
4. **Onboarding Specific**: Add a "Welcome to the Team" section where existing members share one piece of advice for the new hire and one thing they wish they had known on their first day.
5. **Energy Booster (Mid-Workshop)**: Use "Speed Round Questions" — 30 seconds per person, rapid-fire answers. Good for re-energizing a group after lunch or a long session.

## Build Instructions

1. Create board at 3000 x 2500px with Warm White (#FFF8E1) background.
2. Place the Welcome Banner (2900 x 300px) at the top with Deep Purple background.
3. Create 4 activity frames in a 2x2 grid (each 1400 x 900px):
   - Top-left: Fun Questions (yellow background)
   - Top-right: Fact Cards (mint background)
   - Bottom-left: Two Truths (blue background)
   - Bottom-right: Photo Gallery (coral background)
4. Populate Fun Questions frame with 10 question sticky notes in varied colors, plus 3 example answers.
5. Build the Fact Card template as a dashed-border rectangle. Add 3 completed example cards with colored borders.
6. Create the Two Truths grid with 4 player columns, 3 statement sticky notes each, plus hidden answer notes.
7. Set up the Photo Gallery with 5 prompt sticky notes and 5 placeholder image frames with captions.
8. Add the Facilitator Guide (2900 x 400px) at the bottom with activity selection table and suggested combos.
9. Add navigation dots on the Welcome Banner linking to each activity.
10. Leave all participant zones (sticky note areas, card templates, photo frames) unlocked for editing.

## Tips & Best Practices

- The facilitator should always go first to model the expected level of openness and set the tone.
- For introverts, use activities that allow writing first (Fun Questions, Fact Cards) before sharing verbally.
- Do not force participation — offer a "pass" option. Coerced vulnerability destroys trust.
- For remote teams, turn cameras on during icebreakers even if they are usually off — faces build connection.
- Keep icebreakers under 15 minutes unless it is a dedicated team-building session. Energy wanes quickly.
- Avoid icebreakers that create winners and losers (unless it is Two Truths, where losing is fun).
- Save interesting facts from icebreakers and reference them later — "Marcus, you mentioned your punk band — that explains your debugging playlist!"
- Rotate activities across meetings to keep things fresh. Never use the same icebreaker twice in a row.
