# Immersive Conference Map

## Overview

- **Purpose**: A large-scale isometric-style conference venue map that transforms a flat Miro board into a navigable conference experience. Each zone represents a physical area of the venue (main stage, breakout rooms, expo hall, networking lounge, registration) with realistic spatial relationships, pathways between areas, and embedded schedules. Attendees can explore the map to find sessions, speakers, and networking opportunities, creating a visual wayfinding experience that mirrors walking through the actual venue.
- **Best For**: Conference organizers creating digital venue maps, event marketing teams building pre-event excitement, virtual conference platforms needing spatial navigation, hybrid event planners bridging physical and digital experiences.
- **Complexity**: Advanced
- **Board Size**: 5000 x 3500px

## Example Content

"Innovate Summit 2026" — a 3-day B2B tech conference in San Francisco with 3,200 attendees, 48 sessions across 3 tracks, 45 exhibitors, and hands-on workshop labs. Modeled after major tech conferences like SaaStr, Web Summit, and AWS re:Invent.

## Variations

1. **Academic Conference Map**: Replace expo with poster sessions, add peer review rooms
2. **Medical Conference Map**: Add CME credit tracking, pharma exhibit regulations
3. **Developer Conference Map**: Focus on hackathon space, open-source pavilion, code labs
4. **Trade Show Map**: Expand expo hall, reduce sessions, add buyer meeting rooms

## Build Instructions

1. Create board at 5000x3500px with Soft Cream background
2. Place title banner at top with conference branding
3. Position Registration at top-left as the entry point
4. Place Main Stage prominently at top-center
5. Arrange Breakout Rooms, Expo Hall, Networking, and Food Court around the stage
6. Add Workshop Labs as a separate wing
7. Place Schedule and Speaker Directory as reference panels at the bottom
8. Draw pathway connectors between all zones showing physical routes

## Tips & Best Practices

- Embed actual speaker headshots as images on speaker cards
- Link each session card to a detailed session page with abstracts and bios
- Add real-time capacity indicators during the event (green/yellow/red)
- Create a mobile-friendly version by using Miro's presentation mode
- Update the map daily during the conference to reflect schedule changes
- Add a "You Are Here" marker for digital wayfinding on venue screens
