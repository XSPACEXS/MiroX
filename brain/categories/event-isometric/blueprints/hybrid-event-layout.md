# Hybrid Event Layout

## Overview

- **Purpose**: A split-world event board that visualizes both the physical venue and virtual experience side by side, connected by a "bridge" layer showing how the two worlds interact. The left half is the physical venue with rooms, stages, and networking areas. The right half is the virtual platform with streams, chat rooms, and digital activities. The middle bridge shows cameras, livestreams, chat bridges, and shared polling — making the hybrid model tangible and plannable.
- **Best For**: Hybrid event producers managing both physical and virtual audiences, AV teams planning camera and stream setups, event marketers designing engagement for both audiences, corporate event teams running hybrid all-hands or product launches.
- **Complexity**: Advanced
- **Board Size**: 5000 x 3500px

## Build Instructions

1. Create board at 5000x3500px with Light Gray background
2. Place title banner at top
3. Divide the board into three vertical columns: Physical (left), Bridge (center, narrow), Virtual (right)
4. Stack Physical zones vertically: Main Stage, Expo, Networking
5. Stack Virtual zones in mirror layout: Livestream, Digital Expo, Online Networking
6. Create the Bridge column with AV infrastructure connecting corresponding zones
7. Add Engagement Metrics dashboard as a wide footer
8. Draw connectors through the Bridge layer connecting physical to virtual

## Tips & Best Practices

- Run a tech rehearsal 48 hours before the event to test all bridge connections
- Assign a dedicated "virtual experience producer" who monitors the virtual side
- Use different colored name badges: blue for physical, green for hybrid mixer volunteers
- Record all sessions from the virtual platform as backup even if physical AV fails
- Debrief with both physical and virtual attendees separately — their experiences differ
- Plan for 2x the technical support for virtual vs physical
