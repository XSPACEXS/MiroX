# Content Guide — Event Guide Immersive Isometric

## Domain Knowledge: Event Information Architecture

### What Belongs on an Immersive Event Board

An immersive event guide must serve two masters: spectacle (the visual world) and utility (the practical information). The content must be rich enough to be genuinely useful while concise enough to fit within the visual zones of the isometric world.

**The eight content categories for event boards:**

1. **Event Identity**: Name, dates, location, tagline, branding, statistics
2. **Venue Map**: Zones, rooms, stages, outdoor areas, pathways, capacities
3. **Schedule**: Session times, tracks, speakers, room assignments
4. **Speaker Information**: Names, titles, companies, session details, photos
5. **Logistics**: Registration, WiFi, food, parking, first aid, accessibility
6. **Engagement**: Polls, Q&A, social feeds, gamification, networking activities
7. **Sponsor Information**: Sponsor logos, booth locations, sponsor sessions
8. **Companion/Live**: Real-time updates, current session, next up, announcements

## Writing Style for Event Boards

### Tone

- **Welcoming and energetic**: The board is part of the event experience. It should feel exciting, not bureaucratic.
- **Action-oriented**: "Join us at..." not "There will be a session on..."
- **Practical above all**: Every piece of text should help an attendee make a decision or find something.
- **Inclusive**: Mention accessibility features, dietary accommodations, and quiet zones naturally.

### Length Constraints

| Element           | Max Length    | Example                          |
| ----------------- | ------------- | -------------------------------- |
| Event title       | 30 characters | "Innovate Summit 2026"           |
| Zone title        | 40 characters | "Main Stage — Grand Ballroom"    |
| Session title     | 50 characters | "The Future of AI in Enterprise" |
| Speaker one-liner | 60 characters | "Dr. Sarah Chen, CEO NovaTech"   |
| Sticky note       | 20 words      | Quick, practical tips or alerts  |
| Pathway label     | 8 words       | "Through the lobby, 3 min walk"  |

### Things to Include That People Forget

- WiFi network and password (people will ask 100 times)
- Charging station locations
- Restroom locations (especially for large venues)
- Nearest coffee at any given zone
- Walking time between zones
- Session recording availability
- Dress code / temperature advice
- Social media hashtag
- Photo policy
- Code of conduct reference

## Content Refresh Cadence

### Pre-Event (Building Phase)

| Content Type        | When to Finalize | Owner             |
| ------------------- | ---------------- | ----------------- |
| Event identity      | 8 weeks before   | Marketing         |
| Venue map / zones   | 4 weeks before   | Event ops         |
| Schedule / sessions | 2 weeks before   | Programming       |
| Speaker info        | 2 weeks before   | Speaker relations |
| Logistics           | 1 week before    | Event ops         |
| Sponsor info        | 1 week before    | Partnerships      |

### During Event (Live Updates)

| Content Type     | Update Frequency    | Owner             |
| ---------------- | ------------------- | ----------------- |
| Companion zone   | Every 15-30 minutes | Event coordinator |
| Schedule changes | As they happen      | Programming       |
| Capacity alerts  | As needed           | Venue staff       |
| Social feed      | Continuous          | Social media team |

### Post-Event (Archive)

- Add session recording links (within 48 hours)
- Add photo galleries (within 1 week)
- Add attendee count actuals (within 1 day)
- Add satisfaction survey results (within 2 weeks)
- Lock the board and add "ARCHIVED" label
