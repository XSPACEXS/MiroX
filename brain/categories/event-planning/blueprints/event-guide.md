# Event Guide — Immersive Conference Overview Board

## Overview

- **Purpose**: Provides a bird's-eye spatial map of an entire event or conference, showing all zones, areas, and experiences attendees will encounter. Inspired by large-scale conference overview boards where stakeholders can see the full event landscape at a glance — from registration to main stage to breakout sessions.
- **Best For**: Event organizers, conference producers, marketing teams planning summits, and leadership teams reviewing event scope.
- **Complexity**: Advanced
- **Board Size**: 5000 x 3500px

## Board Layout

The board uses a spatial/map-like layout simulating a real event venue viewed from above. Zones are arranged organically rather than in a rigid grid, giving the feel of walking through the event space. A large title banner sits at the top. Below it, zones are spread across the canvas like a campus map.

## Variations

1. **Single-Day Event**: Remove Day 3 from agenda, consolidate Companion Experiences into main track, reduce Expo Hall to 6 booths.
2. **Virtual Conference**: Replace venue zones with virtual platform zones (Main Stream, Breakout Rooms, Virtual Expo, Chat Lounge, On-Demand Library). Keep spatial layout but label as platform sections.
3. **Hybrid Event**: Add a "Virtual Attendee Experience" frame alongside each physical zone showing the digital counterpart (live stream, virtual networking, digital swag bag).
4. **Community Meetup (Small Scale)**: Use only Registration, Main Stage, Networking, and Community Hub frames. Reduce board to 3000 x 2000px.

## Build Instructions

1. Create the board at 5000 x 3500px with Soft Ivory (#FAFAFA) background.
2. Place the title banner frame at the top spanning the full width (4800 x 500px). Set background to Deep Indigo. Add title text in white, 72pt bold. Add subtitle and tagline below.
3. Create the Registration frame (1200 x 900px) at top-left with Light Lavender background. Add all sticky notes and flow elements.
4. Create the Main Stage frame (1800 x 1000px) at top-center. Add stage shape, seating semicircle, speaker sticky notes, and AV requirements.
5. Create Companion Experiences (1200 x 900px) at top-right with amber-tinted background.
6. Continue with Community Hub (middle-left), Speaker's Showcase (middle-center), and Lightning Talks (middle-right).
7. Place the Agenda overview frame spanning the center width (3600 x 600px). Build the 3-day schedule table with color-coded blocks.
8. Add bottom row: Sponsors & Expo (left), Networking Lounge (center), Quiet Zone (right).
9. Draw all connectors between frames using the styles described in Connectors & Flow.
10. Add a floating legend in the bottom-right corner explaining color coding (blue = keynote, pink = workshop, teal = networking, green = meals/breaks).
11. Lock background elements and frame borders. Leave sticky notes unlocked for collaborative editing.

## Tips & Best Practices

- Use Miro's built-in presentation mode to walk stakeholders through the board zone by zone during planning meetings.
- Assign each zone to a team lead and use Miro comments for async coordination.
- Link this overview board to detailed sub-boards (agenda, speaker showcase, venue layout) using Miro card links.
- Add a "Status" tag to each zone: Planning, Confirmed, In Progress, Complete.
- Use Miro's voting feature during retrospectives to identify which zones had the biggest impact.
- Export the board as a high-resolution PNG for print signage or digital displays at the venue entrance.
- Update the board live during the event to reflect schedule changes, room swaps, or capacity alerts.
