# Event Planning Boards — Variations

## Overview

Events come in enormous variety — from intimate 20-person workshops to 10,000-person
conferences, from formal galas to casual community meetups, from single-room sessions to
multi-venue festivals. This guide provides adaptation guidelines for different event types,
scales, and cultural contexts.

---

## Event Type Variations

### Tech Conference / Industry Summit (500-5,000 attendees)

**Characteristics**: Multi-track agenda, keynote speakers, expo hall, networking events,
breakout sessions, often hybrid with virtual component.

**Adaptations**:

- Use the full Isometric Event World pattern with 8-12 zones
- Include a Speaker Showcase zone with 3-tier card hierarchy
- Add an Expo/Sponsor section with booth layout and tier labels
- Include a hybrid component (virtual livestream, chat bridge) if applicable
- Color palette: Indigo Summit (professional, prestigious)
- Board size: 5000x4000px minimum

### Company All-Hands / Town Hall (100-1,000 attendees)

**Characteristics**: Single venue, internal audience, executive presentations, Q&A,
team activities, company culture focus.

**Adaptations**:

- Simpler zone structure: 4-6 zones (Main Stage, Breakout Rooms, Dining, Social)
- Replace "Speakers" with "Presenters" — internal titles and teams, not external bios
- Add a "Company Updates" section with department highlights
- Include team-building activity zones
- Add a "Q&A Board" zone where employees can submit questions in advance
- Color palette: Tech Campus (clean, modern, company-branded)
- Board size: 4000x3000px

### Meetup / Community Event (20-200 attendees)

**Characteristics**: Informal, single-track, community-driven, often recurring, minimal
production requirements.

**Adaptations**:

- Skip the isometric world — use a simple 3-4 frame layout
- Combine agenda and venue into a single frame
- Speaker section is a compact list, not a gallery
- Add a "Community Board" zone for attendee introductions or topic suggestions
- Focus on warmth and accessibility over visual grandeur
- Color palette: Festival Warmth (inviting, community-focused)
- Board size: 3000x2000px

### Gala / Award Ceremony (200-2,000 attendees)

**Characteristics**: Formal, evening event, high production value, entertainment,
dining, VIP management, presentations/awards.

**Adaptations**:

- Use the Night Gala palette (dark backgrounds, gold accents)
- Zone structure: Reception, Cocktail Hour, Main Ballroom, Award Stage, Dining, Lounge
- Speaker section replaced by "Presenters & Performers" with entertainment lineup
- Add a "Seating Chart" zone for table assignments
- Include a "Production Cue Sheet" zone for stage management
- Pathways use champagne gold on dark backgrounds
- Decorative elements: Gold stars, chandelier shapes, elegant dividers
- Board size: 5000x3500px

### Workshop / Training Session (10-50 attendees)

**Characteristics**: Interactive, facilitator-led, materials-driven, timed activities,
breakout groups.

**Adaptations**:

- This overlaps with category 08-Workshops — consider using workshop templates instead
- If using an event board: Simple 2-3 zone layout (Welcome, Main Room, Breakout Area)
- Focus on the activity schedule rather than spatial zones
- Include materials checklist (handouts, supplies, tech requirements)
- Add facilitator notes and timing guides
- Board size: 3000x2000px

### Festival / Multi-Venue Event (1,000-10,000+ attendees)

**Characteristics**: Multiple stages, outdoor areas, food vendors, art installations,
VIP areas, camping/accommodations, extensive logistics.

**Adaptations**:

- Expand to 12-15 zones including: Main Stage, Second Stage, Food Court, Art Walk,
  VIP Lounge, Kids Area, Camping, First Aid, Information Tent, Parking
- Use a larger board: 7000x5000px or more
- Pathways become more complex — use a main loop with branches
- Add weather contingency annotations ("Rain plan: Move to Tent B")
- Include food vendor map with cuisine type labels
- Add safety zones (First Aid, emergency exits, security checkpoints)
- Add a "Festival Map" that could be printed for attendee distribution
- Color palette: Festival Warmth with bold accent colors for each stage

### Virtual Conference (500-10,000+ attendees)

**Characteristics**: Entirely online, multiple virtual "rooms", chat-based networking,
on-demand content, global time zone consideration.

**Adaptations**:

- Replace physical zones with platform zones: Main Stream, Breakout Rooms, Virtual Expo,
  Chat Lounge, On-Demand Library, Networking Hall
- The "spatial" layout represents the virtual platform structure, not a physical venue
- Add time zone indicators for all sessions
- Include technology requirements (browser, bandwidth, platform login)
- Add an "Engagement Features" zone: polls, chat, Q&A, emoji reactions, virtual swag
- Pathways become "navigation links" — how users move between platform sections
- Color palette: Tech Campus (clean, digital-native)

## Scale Variations

### Small Scale (Under 100 attendees)

- Reduce to 3-5 zones
- Single-track agenda (no parallel sessions)
- Speaker list instead of gallery
- Combined registration + welcome zone
- Skip the expo/sponsor section unless there are sponsors
- Board size: 3000x2500px

### Medium Scale (100-500 attendees)

- Standard 6-8 zones
- 2-3 parallel tracks
- Speaker gallery with 2 tiers (keynote + featured)
- Dedicated registration zone
- Expo section with 4-8 booths
- Board size: 4500x3500px

### Large Scale (500-2,000 attendees)

- Full 8-12 zones with the Isometric Event World pattern
- 3-5 parallel tracks with dedicated rooms
- Speaker gallery with 3 tiers
- Large expo hall with booth map
- Multiple networking areas
- Dedicated accessibility information
- Board size: 5000x4000px

### Mega Scale (2,000+ attendees)

- Board-of-boards architecture:
  - Hub board: Isometric Event World overview
  - Sub-boards: Detailed Agenda, Full Speaker Directory, Venue Floor Plan, Expo Map
- 5+ parallel tracks
- Multiple venue floors
- Complex logistics (shuttle buses, parking, multiple entry points)
- Dedicated safety and emergency planning
- Board size: Hub at 6000x5000px, sub-boards at 5000x4000px each

## Cultural and Regional Considerations

### Western Corporate (US, UK, EU)

- Professional tone, structured layout, clear information hierarchy
- Emphasis on networking opportunities and business value
- Speaker bios highlight professional achievements and company affiliation
- Sustainability messaging (carbon offset, digital-first materials)

### APAC / Asian Markets

- Consider right-to-left or vertical text for Japanese/Chinese content
- Group dynamics over individual networking — highlight group sessions
- Speaker bios may emphasize educational credentials and institutional affiliations
- Consider cultural color associations (red is auspicious in Chinese culture,
  white is associated with mourning in some Asian cultures)

### Middle East / North Africa

- Include prayer room location prominently
- Ensure dietary accommodation details are comprehensive (halal certification, etc.)
- Gender-specific considerations for some events
- Consider right-to-left text layout for Arabic-language boards
- Use green and gold as culturally positive colors

### Latin America

- Emphasize social and community aspects — networking zones should be prominent
- Longer meal breaks are culturally expected and should be visually prominent
- Warm color palettes align with cultural aesthetic preferences
- Entertainment and music should be highlighted
