# Brand Guidelines Board

## Overview

- **Purpose**: Document and communicate the complete brand identity system in a visual, shareable format. Serves as the single source of truth for logo usage, color palette, typography, tone of voice, imagery style, and do's and don'ts for anyone creating brand-related materials.
- **Best For**: Brand managers, design teams, external agencies, new hires, and any stakeholder who needs to produce on-brand materials.
- **Complexity**: Simple
- **Board Size**: 4200x3400px

## Connectors & Flow

- No directional flow connectors — this is a reference board, not a process board.
- Thin horizontal divider lines (#E0E0E0, 1px) separate the three row-sections visually.
- The Brand Overview frame (Frame 1) anchors the top with a strong blue visual bar.

## Example Content

The board is pre-filled with a complete brand identity system for "AutoPilot," a fictional SaaS product. Every element contains specific, actionable guidance: hex codes with RGB/CMYK equivalents, real type scale with exact sizes and weights, four tone-of-voice attributes with good/bad examples, and comprehensive do's and don'ts across logo, color, typography, and tone. The examples are realistic enough to use as a starting template for any B2B SaaS brand.

## Variations

1. **Consumer Brand (B2C)**: Add a "Brand Mascot" frame with character guidelines. Expand the color palette to be more vibrant. Add a "Packaging" section with physical product guidelines.
2. **Sub-Brand / Product Line Extension**: Add a frame showing the relationship between the parent brand and sub-brand. Include co-branding rules and lockup configurations.
3. **Social Media Brand Kit**: Add platform-specific profile image specs, cover photo templates, post templates for each platform, and hashtag guidelines.

## Build Instructions

1. Create board at 4200x3400px with background #FFFFFF.
2. Place Frame 1 (Brand Overview) at (50, 50), size 4100x500, background #2962FF.
3. Place Frame 2 (Logo) at (50, 600), size 1800x1000. Place Frame 3 (Color) at (1900, 600), size 2200x1000.
4. Place Frame 4 (Typography) at (50, 1650), size 1800x1000. Place Frame 5 (Tone) at (1900, 1650), size 2200x1000.
5. Place Frame 6 (Imagery) at (50, 2700), size 1800x1000. Place Frame 7 (Do's/Don'ts) at (1900, 2700), size 2200x1000.
6. Add left-border color accents to each frame.
7. Populate color swatches as circles with labels beneath.
8. Populate typography scale as actual styled text blocks at their specified sizes.
9. Use green sticky notes for "Do" items and red for "Don't" items.
10. Add the blue tint gradient strip in the Color Palette frame.
11. Add thin horizontal dividers between row sections.

## Tips & Best Practices

- This board should be view-only for most team members — restrict editing access to the brand manager and design lead.
- Link to this board from every creative brief and design project board so it is always one click away.
- Update the version number and date whenever changes are made (currently v2.1, March 2026).
- Export the board as a PDF for external agencies who may not have Miro access.
- Review and refresh the brand guidelines annually — brands evolve, and outdated guidelines lead to drift.
