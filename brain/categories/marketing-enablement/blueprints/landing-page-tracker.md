# Landing Page Tracker

## Overview

- **Purpose**: Visual tracker for managing the full lifecycle of landing pages — from wireframe concepts through copy variations, design iterations, A/B test results, and conversion optimization. Each landing page gets a dedicated lane showing its current state, performance metrics, and optimization pipeline.
- **Best For**: Growth marketing teams managing multiple landing pages, conversion rate optimization specialists, product marketing teams launching feature-specific pages, agencies delivering landing page projects to clients.
- **Complexity**: Advanced
- **Board Size**: 5000 x 3500px

## Frames & Sections

### Frame 1: Title & Dashboard Summary

- **Size**: 4800 x 350px | **Background**: Deep Navy (#2C3E50)
- "Landing Page Tracker" (56pt), active pages count, avg conversion rate, total A/B tests running

### Frame 2: Wireframe Pipeline

- **Size**: 1400 x 1000px | **Background**: Slate Blue at 10%
- 6 wireframe cards: Homepage v4, Pricing Redesign, AI Assist LP, Enterprise LP, Webinar Registration, Free Trial
- Each card: page name, wireframe status, stakeholder feedback, target launch date

### Frame 3: Copy Variations Lab

- **Size**: 1400 x 1000px | **Background**: Amber at 10%
- Headline A/B pairs for each active page with character counts
- Value prop variations, CTA button text options, social proof copy options
- "Copy scoring: Headline A (clarity 9/10, urgency 7/10) vs Headline B (clarity 7/10, urgency 9/10)"

### Frame 4: Design Iterations Gallery

- **Size**: 1400 x 1000px | **Background**: Purple at 10%
- Visual mockup placeholders for each landing page version
- Design checklist: Mobile responsive, ADA compliant, brand guidelines, load speed
- Feedback stickies from stakeholder reviews

### Frame 5: A/B Testing Control Center

- **Size**: 2400 x 900px | **Background**: Teal at 8%
- Active tests grid: Test name, variant A vs B, metric tracked, sample size, confidence level, days running
- "Pricing page: Variant B +18% conversion, 97% confidence, 14 days — WINNER DECLARED"
- "AI Assist LP: Variant A +6% but only 72% confidence — needs 5 more days"
- Statistical significance calculator reference

### Frame 6: Live Pages Performance

- **Size**: 2400 x 800px | **Background**: Green at 8%
- Performance table: Page name, monthly visits, conversion rate, bounce rate, avg time on page
- Homepage: 85K visits, 3.2% conv, 42% bounce | Pricing: 34K visits, 8.1% conv, 28% bounce
- Top performer and underperformer callouts

### Frame 7: Optimization Backlog

- **Size**: 1500 x 700px | **Background**: Coral at 10%
- Prioritized list of optimization ideas sorted by impact vs effort
- "Add video testimonial to AI Assist LP — Est. +15% conversion, Medium effort"
- "Simplify enterprise form from 8 to 4 fields — Est. +25% completion, Low effort"

## Connectors & Flow

1. Wireframes --> Copy Lab --> Design Gallery (labeled "Content pipeline")
2. Design Gallery --> A/B Testing (labeled "Launch tests")
3. A/B Testing --> Live Pages (labeled "Winners go live")
4. Live Pages --> Optimization Backlog (labeled "Performance informs next tests")
5. Optimization Backlog --> Wireframes (labeled "Cycle continues")

## Example Content

12 landing pages tracked across full lifecycle. Key metrics: avg 5.4% conversion rate, 4 active A/B tests, 2 winners declared this month, $840K pipeline attributed to landing pages.

## Variations

1. **E-commerce Product Pages**: Replace landing pages with product detail pages, add cart conversion metrics
2. **SaaS Signup Flow**: Track multi-step signup flow with drop-off at each step
3. **Event Registration Pages**: Focus on registration conversion with countdown timers and early bird pricing tests

## Build Instructions

1. Create board 5000x3500px, Soft White background
2. Place title banner, then create pipeline stages left-to-right
3. Create A/B Testing center as a wide frame in the middle
4. Place Live Pages performance table below
5. Add Optimization Backlog as feedback loop
6. Connect all stages with flow arrows

## Tips & Best Practices

- Screenshot each landing page version and embed as images in the Design Gallery
- Link A/B test cards to your analytics tool (Google Optimize, Optimizely, VWO)
- Celebrate winners publicly — mark them with green "WINNER" badges
- Archive losing variants but keep them for reference — they inform future tests
- Review this board weekly with the growth team
